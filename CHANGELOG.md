# سجل التغييرات (Changelog)

> **قاعدة الترقيم**: MAJOR.MINOR.PATCH — حيث MAJOR للتغييرات الجذرية، MINOR للميزات الجديدة، PATCH للإصلاحات والتحسينات.

## [2.0.8] - 2026-07-10

### الأمان والثغرات (Security & Vulnerability Fixes)
- 🛡️ **حرج**: إصلاح `UnmarshalJSON` في `money.go` حيث كانت الأعداد الصحيحة تُقرأ كـ cents بدلاً من units
- 🛡️ **حرج**: نقل مفاتيح Google OAuth من `const` في الكود إلى `secureconfig` المشفر
- 🛡️ **عالي**: تمرير Gemini API key عبر HTTP header بدلاً من URL query string
- 🛡️ **عالي**: تشفير GeminiAPIKey في `AppPreferences` في SQLite
- 🛡️ **عالي**: توحيد سلوك `VerifyAdminPin` في جميع المسارات
- 🛡️ **عالي**: تدوير مفاتيح Supabase وتنظيف git history
- 🛡️ **متوسط**: توحيد ثوابت الصلاحيات في `domain/permissions.go` وإزالة التكرار من `pkg/auth/`
- 🛡️ **متوسط**: إصلاح حسابات الأقساط لاستخدام `Amount` بدلاً من `float64`
- 🛡️ **متوسط**: إزالة استخدام `any` من كود الواجهة الأمامية
- 🛡️ **منخفض**: تجميع (precompile) الـ regex في `staff_service.go`

### التحسينات (Improvements)
- 📦 نظام صور المنتجات: ترحيل Base64 من قاعدة البيانات إلى نظام ملفات (Filesystem) مع خادم صور محلي
- 🖼️ تمكين Mica backdrop في Windows 11 للنوافذ الشفافة
- 🔄 إضافة `KeepAliveSupabase` لمنع تعليق مشروع Supabase
- 🌐 دعم الترجمة i18n مع توسيع التغطية النصية
- 🎯 تحسين أداء الشبكة الافتراضية (VirtualProductGrid) مع TanStack Virtual
- 📊 تحسين دقة الحسابات المالية باستخدام نوع `Amount` (int64 cents)

### الإصلاحات (Bug Fixes)
- إصلاح Race Condition في الـ DB transactions أثناء عمليات البيع المتزامنة عبر LAN
- إصلاح فقدان بيانات النافذة (position/size) عند الإغلاق غير الطبيعي
- إصلاح مشكلة تعليق التطبيق عند فشل اتصال Supabase
- إصلاح تسرب الذاكرة في بث Gemini API stream
- إصلاح عدم تحديث أرصدة العملاء بعد الدفع في وضع Offline
- إصلاح عدم تطابق حقول JSON بين Go backend و TypeScript frontend

## [2.0.6] - 2026-05-20

### الميزات الجديدة (Features)
- 🌐 إضافة LAN Server مع UDP discovery و Heartbeat protocol
- ☁️ إضافة النسخ الاحتياطي السحابي عبر Supabase (تقسيم تلقائي للملفات الكبيرة)
- 💳 إضافة Split Payments (تقسيم الفاتورة على أكثر من طريقة دفع)
- 📋 إضافة Purchase Orders (دورة حياة كاملة لأوامر الشراء)
- 🔐 إضافة تسجيل الدخول بالـ PIN السريع مع خيار `MustChangePin`

### التحسينات (Improvements)
- 🏗️ إعادة هيكلة المشروع إلى Clean Architecture (Handlers → Service → Repository)
- 🔄 استبدال CGO sqlite3 بمكتبة glebarez/sqlite النقية (لا حاجة لـ GCC)
- 📝 تحسين توليد فواتير PDF مع دعم QR code والباركود

### الإصلاحات (Bug Fixes)
- إصلاح خطأ فقدان البيانات عند إغلاق التطبيق أثناء عملية بيع
- إصلاح مشكلة تعدد نسخ التطبيق (Single Instance Lock)
- إصلاح خطأ في حساب نقاط الولاء للعملاء
